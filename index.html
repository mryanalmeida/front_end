One Page Site

Ideal para negocios de pequeno ou médio porte o site One Page é site onde você inclui todas as informações que o visitante precisa de forma objetiva e criativa em uma única pagina.

Wireframe

Wireframe é uma representação visual de estrutura básica de uma pagina. Ele deve ser feito de maneira mais simples possível, mostrando apenas o essencial, como uma espécie de rascunho, sem cores ou imagens. Clique no botão ou escaneie o QR code para obter o wireframe desta pagina
